# CLAUDE.md — Клініка імені Святого Пантелеймона

## Проект
Landing page для приватної медичної клініки "Кабінет педіатра" в Калуші.

---

## Актуальні дані клініки

| Поле | Значення |
|------|----------|
| **Назва** | Клініка імені Святого Пантелеймона |
| **Адреса** | вул. Богдана Хмельницького 1, м. Калуш, Івано-Франківська обл. |
| **Телефон** | +38 (097) 699-53-84 |
| **Графік** | Пн-Пт 09:00-17:00, Сб-Нд — вихідний |
| **Google Maps** | https://maps.app.goo.gl/XvBrNRqV7z7TcpAk7 |

---

## Технічний стек

- **HTML5** — семантична розмітка
- **Tailwind CSS** — CDN версія
- **Vanilla JavaScript** — без фреймворків
- **Google Fonts** — DM Serif Display + DM Sans

## Архітектура коду

```
/clinic-landing/
├── index.html           # Єдиний файл (HTML + CSS + JS)
├── CLAUDE.md            # Цей файл
└── DESIGN_GUIDELINES.md # Принципи дизайну
```

### JavaScript модулі (в index.html)

| Модуль | Опис |
|--------|------|
| `CONFIG` | Константи (інтервали, тривалості) |
| `Utils` | Утиліти (storage, селектори, debounce) |
| `LanguageManager` | Мультимовність UK/EN |
| `ClinicStatus` | Статус відкрито/закрито |
